# Project Breakdown
This project consists of three sections:
### Wordsearch Generator
This section is responsible for creating the word search puzzles. To learn more about this view 
`wordsearch_generator/README.md`.
### Wordsearch Solver
This section is responsible for solving the word searches created in the `wordsearch_generator` using deep learning models.
For more information view `wordsearch_solver/README.md`.
### Wordsearch Web App
This section is responsible for providing a web interface to generate and solve word searches. For more information view `web_app/README.md`.
